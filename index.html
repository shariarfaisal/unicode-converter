📥 Installation Guide

This extension is not published on the Chrome Web Store yet. Follow these simple steps to install it manually.

Step 1: Download the Extension

  1. Click the "Download Extension" button above or visit GitHub repository
  2. On GitHub, click the green "Code" button → "Download ZIP"
  3. Extract the ZIP file to a folder on your computer (e.g., Desktop or Documents)
  4. You should see a folder named unicode-converter-main

Step 2: Load the Extension in Chrome

  1. Open Google Chrome browser
  2. Type chrome://extensions/ in the address bar and press Enter
  3. Enable "Developer mode" by toggling the switch in the top-right corner (it will turn blue)
  4. Click the "Load unpacked" button that appears
  5. Navigate to the extracted folder and select the unicode-converter-main folder
  6. Click "Select" or "Open"

Step 3: Verify Installation

  1. ✅ Look for "Unicode Text Formatter v2.0.0" in your extensions list
  2. ✅ You should see a purple "A" icon in the Chrome toolbar
  3. ✅ Click the puzzle icon (🧩) in Chrome toolbar to find your extensions
  4. ✅ Click the pin icon next to "Unicode Text Formatter" for easy access
  5. ✅ Click the extension icon to open the popup and start using it!
💡 Important Note:

The extension requires "Developer mode" to be enabled since it's not published on the Chrome Web Store. This is completely safe and normal for locally installed extensions. Your data stays private and secure on your computer.

🔧 Troubleshooting

Extension icon not showing?

Click the puzzle icon (🧩) in Chrome toolbar, find "Unicode Text Formatter" and click the pin icon to add it to your toolbar.

Tooltip not appearing when I select text?

Make sure the toggle is enabled in the extension popup. Also try refreshing the page (F5 or Cmd+R). Check that the extension is enabled in chrome://extensions/.

Text not replacing in input field?

Make sure the input field is focused (clicked). Some websites may prevent text modification. Try using the extension popup instead to copy the formatted text manually.

Need more help?

Visit the complete documentation or report an issue on GitHub.
